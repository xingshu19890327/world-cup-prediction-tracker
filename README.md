# 2026 世界杯预测追踪器

一个类似 Excel tracker 的网页版本，用于记录 2026 世界杯每场比赛的赛程、胜平负预测、比分预测、Sportsbet 赔率、实际比分，以及比分/赛果是否命中。

## 如何运行

```bash
npm install
npm run dev
```

## 如何 build

```bash
npm run build
```

## 如何使用

- 点击 **新增比赛** 创建新行，或直接在主表中编辑已有比赛。
- 双击单元格或点击“弹窗”可打开编辑弹窗。
- 在 **实际比分** 列填写 `2-1` 或 `2:1` 格式。
- 点击 **更新结果 / 重新计算**，系统会自动计算实际胜负、比分命中、赛果命中。
- 使用 **导入 JSON** 导入完整 104 场或后续维护数据。
- 使用 **导出 JSON** 备份当前数据。
- 使用 **导出 CSV** 导出可放入 Excel 的表格。
- 删除比赛时会弹出二次确认。

## 数据保存

第一版没有后端，所有数据保存在浏览器 `localStorage` 中。建议定期导出 JSON 作为备份。

## 自动刷新与赔率

本项目不会自动刷新、不会后台轮询，也不会自动抓取 Sportsbet 赔率。赔率和实际比分需要用户手动录入或通过 JSON 导入。
